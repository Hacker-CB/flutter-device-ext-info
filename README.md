# DeviceExtInfo 

Extended device info flutter plugin.

##Usage:

```dart
void main(){
    final deviceName =  await DeviceExtInfo.deviceName;
    final deviceClass =  await DeviceExtInfo.deviceClass;
    final screenSizeInches =  await DeviceExtInfo.screenSizeInches;
}
```
